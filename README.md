# ILI9341-TouchGFX-STM32F411-HAL

 - **STM32F411CE6** (BackPill)
 - TFT 2.8'' **ILI9341** (*SPI1*) + Touch **XPT2046** (*SPI2*)
 - STM32CubeIDE + TouchGFX Designer 4.19.1
